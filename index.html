1. Ключевые AI-навыки и практические кейсы

1.1 Промт инжиниринг

Правила и приемы написания промтов

Освоенные навыки:

  • Формулировка промптов с учетом роли, задачи, контекста, ограничений и желаемого формата ответа.
  • Применение приемов усиления промптов: уточнения, примеры, шкалы, стиль (tone of voice), структура
  • Оптимизация взаимодействия с нейросетью для получения точных, релевантных и практически применимых результатов

Пример кейса:

Анализ причин низкой маржинальности платной медицинской компании и разработка плана из 5 шагов для их устранения.

Без нейросети: 4-8 часов (сбор данных, расчеты, анализ, формулировка рекомендаций)

С нейросетью: 10-30 минут (ввод промта, проверка и корректировка результата)

Экономия времени: 85-95%

1.2 Промт-инжиниринг

Техники написания промтов

Освоенные навыки:

  • Умение адаптировать промпт под задачу. Знаю, какая техника лучше подходит для разных типов задач — от генерации простого ответа до пошагового решения комплексной задачи или диалога с уточняющими вопросами.
  • Навык взаимодействия с нейросетью как с инструментом решения профессиональных задач.
  • Выстраиваю промпты так, чтобы получать структурированные, полезные и применимые в работе результаты: стратегии, планы, аналитику, автоматизационные решения и пр.

Пример кейса:

Разработка комплексного финансового анализа и рекомендаций для медицинской и торговой компаний, включая SWOT-анализ, план работы финансового директора, выявление причин низкой маржинальности, анализ отчетов о прибылях и убытках, а также предложение мер по оптимизации затрат и увеличению прибыли.

Без нейросети: 8-12 часов

С нейросетью: 1-2 часа

Экономия времени: 85-90%

1.3 Сила последовательных промтов

Решение сложных задач с помощью последовательных промтов

Освоенные навыки:

  • Разделение сложной задачи на этапы с формулировкой последовательных промтов.
  • Использование нейросетей для пошагового решения профессиональных задач.
  • Проверка, уточнение и адаптация ответов нейросети под конкретный бизнес-контекст.

Пример кейса:

Анализ финансовых данных компании по продаже автоматических воротных систем для выявления ключевых статей затрат, определения наиболее растущих расходов, выяснения причин их роста и предложения мер по оптимизации.

Без нейросети: 8-12 часов

С нейросетью: 30-60 минут

Экономия времени: 85-90%
